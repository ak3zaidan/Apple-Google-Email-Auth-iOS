# Apple-Google-Email-Auth-iOS
Fully integrated Apple/Google/Email login for Firebase Auth + Apple Keychain

> Features
- Apple Firebase Auth with Firebase Database (Signup/Login)
- Google Firebase Auth (Signup/Login)
- Email Firebase Auth (Signup/Login)
- Apple Face ID User verificatiom
- Apple keychain credential storage + auto loading
- Clean UI - Easy to modify
